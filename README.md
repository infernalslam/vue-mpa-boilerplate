# Vuejs(MPA) with NodeJS & Nunjucks view engine

## Development setup
* Node Version: v10.15.3+ NPM Version: 6.9.0

install npm package
```
$ npm i
```

run server (when not hot reloading for frontend).
```
$ npm run dev
```

## Directory Structure

* bin
* public
* routes
    * layouts
    * pages
    * views
* src
